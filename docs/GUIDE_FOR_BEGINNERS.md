# OpenClaw VK Plugin — Полный гайд для начинающих

> Этот документ написан для тех, кто никогда не работал с программированием или AI-инструментами.  
> Здесь объясняется **что делает каждая часть системы** и **зачем она нужна** — простым языком.

---

## Что такое OpenClaw вообще?

Представь, что у тебя есть сообщество ВКонтакте — магазин, блог, студия, или просто личная страница.  
OpenClaw — это **бот**, который живёт в твоём сообществе и отвечает на сообщения.

Но это не обычный бот с заготовленными ответами. Он подключён к **настоящим AI** — ChatGPT, Claude, DeepSeek и другим — и может:

- Отвечать на любые вопросы от имени твоего сообщества
- Генерировать картинки по описанию
- Помнить твои заметки и напоминать о делах
- Спрашивать одновременно несколько AI и давать взвешенный ответ

---

## Из чего состоит система

Система разбита на **независимые модули**. Ты устанавливаешь только то, что нужно именно тебе.

---

### Модуль 1 — VK Бот (основа, обязательный)

**Что делает:**  
Это сердце всей системы. Бот "слушает" входящие сообщения в твоём VK-сообществе и отвечает на них. Показывает меню с кнопками, принимает нажатия, запускает нужные функции.

**Как выглядит для пользователя:**  
Пишешь боту в сообщество → он отвечает меню с кнопками → нажимаешь кнопку → получаешь результат.

**Что нужно:**  
Только токен твоего VK-сообщества (бесплатно, получается за 2 минуты — см. ниже).

**Без этого модуля** ничего остального не работает.

---

### Модуль 2 — LLM Прокси (Groq + OpenRouter)

**Что делает:**  
Когда ты задаёшь боту вопрос, ему нужно обратиться к какому-то AI. AI-сервисы дают бесплатный доступ, но с ограничением: один запрос в минуту. Прокси **чередует несколько ключей** так, чтобы это ограничение не ощущалось.

**Простая аналогия:**  
Представь, что ты хочешь позвонить на горячую линию, но линия занята. Прокси — это как иметь 4 телефона: если первый занят, сразу набирается второй.

**Что нужно:**  
- Ключи Groq (бесплатно, регистрация на сайте, занимает 3 минуты)
- Опционально: ключи OpenRouter (тоже бесплатно, добавляет больше AI-моделей)

**Без этого модуля:** бот будет отвечать медленнее или ждать между запросами. Если у тебя мало запросов — можно обойтись без прокси.

---

### Модуль 3 — Browser LLM Bridge (Мост к браузерным AI)

**Что делает:**  
Это самая необычная часть системы. На сервере запускается **настоящий браузер** (Chromium), который открывает сайты ChatGPT, Claude, DeepSeek, Perplexity, Mistral, Qwen — точно так же, как ты бы открывал их в браузере сам. И задаёт вопросы от твоего имени.

**Зачем это нужно:**  
Официальный API ChatGPT стоит деньги. Официальный Claude — тоже. Но **веб-версии этих сервисов бесплатны**. Этот модуль даёт доступ к лучшим AI бесплатно, потому что использует их веб-интерфейс, а не платный API.

**Что нужно:**  
Только один раз залогиниться в каждый сервис через браузер на сервере. Потом сессии сохраняются.

**Важно:** Для этого нужен сервер с ~2 ГБ оперативной памяти на каждый AI-сервис. Итого ~4-6 ГБ для всех шести.

---

### Модуль 4 — Великий Мудрец 🧙

**Что делает:**  
Это самая умная функция системы. Когда ты задаёшь вопрос, Великий Мудрец делает следующее:

1. **Одновременно** спрашивает все 6 AI (ChatGPT, Claude, DeepSeek, Perplexity, Mistral, Qwen)
2. Берёт ключевые части каждого ответа
3. Отдаёт это седьмому AI (Cerebras), который **синтезирует итоговый ответ** — находит общее мнение, отмечает расхождения

**Результат:** ты получаешь не мнение одного AI, а **взвешенный консенсус** нескольких независимых систем. Это намного надёжнее, чем спрашивать одного.

**Простая аналогия:**  
Как если бы ты задал один вопрос сразу шести экспертам, а потом нейтральный модератор собрал их мнения и дал тебе сводку.

**Что нужно:**  
- Модуль 3 (Browser Bridge) — чтобы было откуда брать ответы
- Ключ Cerebras (бесплатный) — для финального синтеза

**Дополнительные возможности:**  
- Сохраняет все обсуждения в историю (можно вернуться к любому)
- Архив старых обсуждений
- Экспорт в разные форматы:
  - 📄 **Текст** — простой текстовый документ
  - 📊 **Таблица** — сравнительная таблица: что ответил каждый AI
  - 📈 **График** — визуальная диаграмма с цветами по каждому AI
  - 🔵 **Диаграмма** — круговая диаграмма консенсуса

---

### Модуль 5 — Личные инструменты

Набор простых утилит, которые работают прямо в боте. Ничего лишнего, всё хранится на твоём сервере.

#### 📝 Заметки
Сохраняй любые тексты через бота. Потом ищи по ключевым словам.  
*Пример: "Запомни: пароль от WiFi — 12345"*

#### ⏰ Напоминания
Ставь напоминалки с датой и временем. Бот напишет тебе в VK в нужный момент.  
*Пример: "Напомни 25 декабря поздравить маму"*

#### ✅ Задачи (Todo)
Список дел с отметками выполнено/не выполнено.  
*Пример: "Добавь в список: купить молоко"*

#### 📰 Ежедневный дайджест
Каждое утро бот присылает сводку: какие задачи открыты, какие напоминания сегодня, краткая сводка.

#### 🔍 Парсер VK
Собирает посты и комментарии из любых VK-групп по заданным ключевым словам. Полезно для анализа конкурентов или мониторинга темы.

**Что нужно:** Ничего дополнительного. Всё хранится в базе данных на твоём сервере.

---

### Модуль 6 — Генерация изображений 🎨

**Что делает:**  
Генерирует картинки по текстовому описанию прямо в боте.

*Пример: "Нарисуй закат над горами в стиле аниме"*

**Доступные стили:**
- **Flux** — лучшее качество, реалистичный
- **SDXL** — Stable Diffusion XL, универсальный
- **DreamShaper** — творческий, фантастический
- **Realistic Vision** — максимально реалистичный
- **Anime** — аниме-стиль

**Что нужно:**  
Ключ ModelsLab (бесплатно, 100 картинок в день).

## Как получить каждый ключ — пошаговые инструкции

### VK Токен сообщества (обязательный)

1. Зайди в своё VK-сообщество
2. Нажми **"Управление"** (справа под обложкой)
3. Перейди в **"Работа с API"** (в левом меню)
4. Нажми **"Создать ключ"**
5. Поставь галочки: **Сообщения**, **Фотографии**
6. Скопируй ключ — он выглядит как длинная строка букв и цифр

**Ошибки:** Если бот не отвечает — убедись, что в настройках сообщества включены **Сообщения** (Управление → Сообщения → Включить).

---

### Groq API ключ (для LLM Прокси, бесплатно)

Groq — это сервис, который даёт быстрый бесплатный доступ к AI (Llama 3, Mixtral и другие).

1. Зайди на [console.groq.com](https://console.groq.com)
2. Нажми **"Sign Up"** (Зарегистрироваться)
3. Можно войти через Google
4. После входа нажми **"API Keys"** в левом меню
5. Нажми **"Create API Key"**
6. Дай ключу любое название (например "openclaw-1")
7. Скопируй ключ — он начинается с `gsk_`

**Совет:** Создай 3-4 ключа (можно с одного аккаунта) — это ускорит работу бота.

**Бесплатный лимит:** 1 запрос каждые 62 секунды на ключ. С 4 ключами — практически мгновенно.

---

### OpenRouter ключ (дополнительный, бесплатно)

OpenRouter даёт доступ к десяткам AI-моделей, часть из которых совершенно бесплатна.

1. Зайди на [openrouter.ai](https://openrouter.ai)
2. Нажми **"Sign In"** → войди через Google или GitHub
3. Перейди в раздел **"Keys"** (вверху справа)
4. Нажми **"Create Key"**
5. Скопируй ключ — начинается с `sk-or-v1-`

**Бесплатные модели:** Qwen 3, Gemma 2, Llama 3 и другие. Платить ничего не нужно.

---

### Cerebras ключ (для Великого Мудреца, бесплатно)

Cerebras — это очень быстрый AI для синтеза ответов. Буквально отвечает за доли секунды.

1. Зайди на [cloud.cerebras.ai](https://cloud.cerebras.ai)
2. Нажми **"Sign Up"**
3. Зарегистрируйся (можно через Google)
4. После входа нажми **"API Keys"** в левом меню
5. Нажми **"Create new API Key"**
6. Скопируй ключ — начинается с `csk-`

**Бесплатный лимит:** Очень щедрый — тысячи запросов в день.

---

### ModelsLab ключ (для генерации картинок, бесплатно)

1. Зайди на [modelslab.com](https://modelslab.com)
2. Нажми **"Sign Up"**
3. Зарегистрируйся
4. После входа зайди в **"Dashboard"** → **"API Key"**
5. Скопируй ключ

**Бесплатный лимит:** 100 изображений в день.

## Установка — что происходит за кулисами

Когда ты запускаешь `sudo bash installer/install.sh`, скрипт делает следующее:

1. **Проверяет**, установлены ли Node.js, Python, браузер. Если нет — ставит автоматически.
2. **Спрашивает** какие модули тебе нужны (показывает меню).
3. **Просит ввести** нужные ключи (по одному, с пояснениями).
4. **Копирует файлы** в нужные папки на сервере.
5. **Создаёт systemd-сервисы** — это значит, что бот будет запускаться автоматически при старте сервера и перезапускаться если упадёт.
6. **Запускает** всё и проверяет что работает.

**Сервер?** Тебе нужна виртуальная машина (VPS). Самый дешёвый вариант — ~$5-10 в месяц (например на TimeWeb, Reg.ru, Timeweb Cloud, Hetzner). Нужна Ubuntu 20.04 или новее.

## Часто задаваемые вопросы

**Q: Мне нужен опыт программирования?**  
A: Нет. Установочник задаст все вопросы и сделает всё сам. Тебе нужно только скопировать ключи с нужных сайтов.

**Q: Это безопасно?**  
A: Ключи хранятся только на твоём сервере в файле `.env`. В код они не зашиты и в репозиторий не попадают. Никто посторонний их не видит.

**Q: Можно установить только часть модулей?**  
A: Да, именно для этого система модульная. Хочешь только бота с заметками — ставишь только модули 1 и 5. Хочешь Великого Мудреца — добавляешь 3 и 4.

**Q: Что если я не знаю что такое "сервер"?**  
A: Сервер — это просто компьютер, который работает 24/7 и доступен из интернета. Его арендуют у хостинг-провайдеров за небольшие деньги. Самый простой вариант: зарегистрироваться на [timeweb.cloud](https://timeweb.cloud), создать VPS с Ubuntu, и ты получишь IP-адрес и пароль для подключения.

**Q: Сколько памяти нужно серверу?**  
A: Зависит от модулей:
- Только бот (модуль 1): 512 МБ RAM
- Бот + LLM прокси: 512 МБ RAM
- Бот + Browser Bridge (3 LLM): ~4 ГБ RAM
- Бот + Browser Bridge (все 6 LLM): ~6-8 ГБ RAM

**Q: Браузерный мост — это не против правил?**  
A: Это такой же доступ к сервисам, как если бы ты сам открыл браузер и написал вопрос. Технически это использование веб-интерфейса, а не API. Для личного или коммерческого использования в разумных объёмах — обычная практика.

**Q: Что если один из AI-сервисов не работает?**  
A: Великий Мудрец работает с теми, кто ответил. Если из 6 ответили 4 — синтез делается по 4 ответам. Система не ломается при недоступности отдельных сервисов.

**Q: Можно использовать без VK?**  
A: Сейчас только VK. Архитектура позволяет добавить Telegram, но это потребует доработки.

## Как боту отвечают AI без платных API — подробнее

Когда установлен Browser Bridge, на сервере запущено несколько браузеров. Каждый браузер открыт на одном сайте:

```
Браузер 1 → сайт chat.openai.com (ChatGPT) — залогинен под твоим аккаунтом
Браузер 2 → сайт claude.ai (Claude) — залогинен
Браузер 3 → сайт chat.deepseek.com (DeepSeek) — залогинен
... и т.д.
```

Когда приходит вопрос, бот **вводит текст** в поле ввода этих сайтов — ровно как ты делал бы это руками — и читает ответ. Это называется "browser automation" (автоматизация браузера).

Чтобы это выглядело естественно и не вызывало подозрений у сайтов, движения мыши и скорость ввода текста имитируют человеческие — с небольшими случайными паузами.

## Структура папок после установки

```
/opt/openclaw-vk-plugin/    — основной бот
  .env                      — твои ключи (никому не показывай!)
  src/                      — логика бота
  personas/                 — AI-роли (кем представляется бот)
  scripts/                  — заметки, напоминания, задачи

/opt/browser-bridge/        — мост к браузерным AI
  browser-llm-bridge.js     — основной сервер моста
  sage.py                   — Великий Мудрец
  report-generator.js       — генерация отчётов

/opt/openclaw-sage/         — данные Великого Мудреца
  sage.db                   — история обсуждений (SQLite)
  reports/                  — сгенерированные отчёты (PNG, HTML, MD)

/opt/groq-proxy/            — прокси для ключей
```

## Если что-то пошло не так

**Бот не отвечает:**
```bash
# Посмотри логи
journalctl -u openclaw -f
```

**Browser Bridge не работает:**
```bash
journalctl -u openclaw-bridge -f
```

**Великий Мудрец зависает:**  
Скорее всего один из AI-сайтов изменил интерфейс или требует капчу.  
Проверь: `journalctl -u openclaw-bridge -n 50`

**Ключ не работает:**  
Зайди на сайт сервиса и проверь, не истёк ли ключ. Groq и OpenRouter иногда деактивируют ключи без использования.
